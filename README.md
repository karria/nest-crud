## Description

래티스 백엔드 엔지니어 사전 과제

## Installation

```bash
$ npm install
```

## Running the app

```bash
$ npm run start
```

## Test

```bash
# unit tests
$ npm run test:unit

# e2e tests
$ npm run test:e2e
```
